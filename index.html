Frequently Asked Questions

Everything you need to know about the Fusion SMP network mechanics, ranks, and rules.

How do I acquire and craft Fusion Cores?

You can craft base tier cores natively in the custom interface tables using materials found in-game, or purchase premium keys from our official store. Use the command /fusions in-game to view item formulas and indexing guides.

Do active fusions drop upon player elimination?

Yes. Fusion SMP features a hardcore high-stakes competitive economy. When a player is eliminated, all active bound profile core modules will instantly safely decouple, drop into the world coordinates, and the victim's profile attributes will force-reset back to the baseline 20 HP defaults.

How can I access the Admin Panel if authorized?

Authorized system administrators or operators (OP) can fire up the master management node terminals using the utility command /fusionadmin directly within the in-game chat interface to dispatch components or clear timers globally.
